# Statice

一个用来查生词然后保存到Anki的插件。  
An extension for search words and add them to Anki.  

> [スターチス](https://hanasaku-gift.com/language-of-flower/statice-2) の花言葉は、「途絶えぬ記憶」、「変わらぬ心」です。

## 💐安装

1. 运行 `npm install && npm run build`
2. 用浏览器加载build文件夹
